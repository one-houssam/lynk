# Lynk

Lynk is a simple local network chat app and the first step toward a local social platform.

## Quick Start

1. Install dependencies: `npm install`
2. Start server: `npm start`
3. Open `http://localhost:3000`
4. From another PC on the same LAN, open `http://HOST_PC_IP:3000`

## Contribute

Want to help build a local social platform with us? Contributions are welcome.

You can contribute by:
- Improving UI/UX
- Adding chat features (rooms, reactions, profiles)
- Improving LAN discovery and reliability
- Fixing bugs and polishing docs
